Aplikasi Inventaris & Peminjaman Barang
Sistem Informasi Inventaris dan Peminjaman Barang berbasis **CodeIgniter 3** untuk membantu pengelolaan data barang, anggota, transaksi peminjaman, serta laporan secara terstruktur dan terintegrasi.

Teknologi yang Digunakan:
* PHP (Rekomendasi 7.4, karena CI3 kurang stabil di PHP 8.1+)
* MySQL / MariaDB
* CodeIgniter 3
* Bootstrap (untuk tampilan dashboard)
* XAMPP / Laragon

Fitur Utama
1. Login Sistem
2. Dashboard
3. Manajemen Anggota
4. Manajemen Barang
5. Transaksi Peminjaman
6. Laporan

Instalasi
1. Clone project
2. Buat database di phpMyAdmin
3. Import file `.sql`
4. Edit konfigurasi database: application/config/database.php
5. Jalankan di browser: http://localhost/nama_project
