# **APLIKASI INVENTARIS & PEMINJAMAN BARANG**

Sistem Informasi Inventaris dan Peminjaman Barang berbasis **CodeIgniter 3** untuk membantu pengelolaan data barang, anggota, transaksi peminjaman, serta laporan secara terstruktur dan terintegrasi.<br><br>


## **Teknologi yang Digunakan**

* **PHP 7.4** (Direkomendasikan, karena CI3 kurang stabil di PHP 8.1+)<br>
* **MySQL / MariaDB**<br>
* **CodeIgniter 3**<br>
* **Bootstrap** (UI Dashboard)<br>
* **Laragon**<br><br>

---

## **Fitur Utama**

**1. Login Sistem**<br>
**2. Dashboard**<br>
**3. Manajemen Anggota**<br>
**4. Manajemen Barang**<br>
**5. Transaksi Peminjaman**<br>
**6. Laporan**<br><br>

---

## **Instalasi**

1. Clone project<br>
2. Buat database di phpMyAdmin<br>
3. Import file `.sql`<br>
4. Edit konfigurasi database di:<br>
   `application/config/database.php`<br>
5. Jalankan di browser:<br>
   `http://localhost/nama_project`<br><br>
